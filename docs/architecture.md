# AI Research Copilot Architecture

## Goal

Build a production-grade multi-agent research assistant capable of understanding research papers using Retrieval-Augmented Generation (RAG).

## Tech Stack

Backend:
- FastAPI

Frontend:
- React

Database:
- PostgreSQL

Vector Database:
- ChromaDB

LLM:
- OpenAI

Agent Framework:
- LangGraph

Deployment:
- Docker
