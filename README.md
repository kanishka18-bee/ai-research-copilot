# AI Research Copilot

A production-grade Multi-Agent Retrieval-Augmented Generation (RAG) platform built with FastAPI, LangGraph, and modern LLM technologies.

## Vision

AI Research Copilot enables researchers and students to interact with research papers using intelligent AI agents capable of answering questions, generating summaries, comparing papers, and assisting with literature reviews.

## Tech Stack (Planned)

- Python
- FastAPI
- LangGraph
- LangChain
- OpenAI / Llama
- ChromaDB / PostgreSQL
- React
- Docker
